# 🧪 Radium-Infused 3D Interactive Portfolio

A high-performance, immersive 3D portfolio experience built with **Next.js 16**, **Framer Motion**, and **Canvas2D**. This project transforms a traditional professional portfolio into a "playable" interactive canvas.

![Project Preview](https://github.com/Shashank-Mishra-1/Portfolio-next/raw/main/public/images/Real_asset.png)

## 🚀 Key Features

- **Interactive Radium Background:** A high-performance drawing engine with 4 generative pen styles:
  - 🕸️ **Geometric Web:** Real-time crystalline structure generation.
  - 🧬 **DNA Ribbon:** Velocity-based 3D twisting ribbons.
  - ⚡ **Radium Plasma:** Harsh, jagged toxic energy trails.
  - 🖋️ **Toxic Ink:** Radioactive calligraphy with a uranium core.
- **3D Tri-Fold Book UI:** A physics-based interactive book that unfolds to reveal Resume, Portfolio, and Contact sections.
- **100% Mobile Responsive:** Adaptive layout that transitions from a 3D horizontal fold on desktop to a smooth vertical stack on mobile devices.
- **Glassmorphism Design:** Modern, premium aesthetic with glowing gradients and blurred surfaces.

## 🛠️ Tech Stack

- **Framework:** [Next.js 16 (App Router)](https://nextjs.org/)
- **Animation:** [Framer Motion](https://www.framer.com/motion/)
- **3D Effects:** [Three.js](https://threejs.org/) (Hybrid 2D/3D Architecture)
- **Styling:** [Tailwind CSS](https://tailwindcss.com/)
- **Runtime:** [Turbopack](https://nextjs.org/docs/app/api-reference/turbopack)

## 📦 Local Setup

1. **Clone the repository:**
   ```bash
   git clone https://github.com/Shashank-Mishra-1/Portfolio-next.git
   cd Portfolio-next
   ```

2. **Install dependencies:**
   ```bash
   npm install
   ```

3. **Run the development server:**
   ```bash
   npm run dev
   ```

4. **Build for production:**
   ```bash
   npm run build
   ```

## 🎨 Customization

To swap the "Stamp" image on the contact card:
1. Prepare a square image (Recommended: `80x80px` or `200x200px`).
2. Save it to `/public/images/stamp-image.png`.
3. The UI will automatically blend and apply the "postage" effect.
